# HelpDesk

Sistema skirta įmonės klientų palaikymui ir jų problemų sprendimui. Frontend naudoja Blade šablonus ir Bootstrap. Backend Laravel. Naudoja default laravel autentifikavima.

## Stack

* Laravel
* SASS
* PostgreSQL
